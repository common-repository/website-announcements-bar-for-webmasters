== Description ==
Announcement Bar is a simple, powerful and flexible plugin which allows you to add an informational bar on the top or bottom of your WordPress site to display any announcement or notifications with a call to action button. 

Announcement Bar is a very effective notification bar plugin which adds a beautifully themed bar to the top or bottom of your website to highlight some specific news or announcement with a call to action button. You can use it to put your blog subscription form in front of users and increase daily visitors and user-base.  

There are several colour options available to choose from for your notification bar and displays a toggle button for users to hide or show the bar. You can generate leads and ask your readers to visit a special offer page. Announce special offers or any milestone achieved. 

This plugin was developed and maintained by Iwebslog Labs at http://www.iwebslog.com/ for WordPress Community. To get professional Hosting and Design please contact us.

=Features  =
1. Different colour options to choose from for the Bar colour.
2. Automatically hides after few seconds. Can change this setting in plugin code editor.
3. Different colour options to choose from for the Button Colour.

== Installation ==
1. Upload the full directory into your wp-content/plugins directory.
2. Use your favourite FTP program like CuteFtp, Core FTP, Filezilla.
3. Go to the plugins administration screen in your WordPress Admin and activate the plugin.
4. Go to the left bar and set the options in Announcement Bar Panel. 
 
Your website is ready to go. !!
